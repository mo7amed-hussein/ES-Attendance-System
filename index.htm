Smart Attendance System

Overview

This System provide an automated way for counting people passing through door or passage, and store this value online each time interval , knowing how many people in a hall or room is very usefull for example check hall capacity and emeregeny situations , also it provide an access control via RFID tobe used in secure places or halls,finally it porvide motion detection that could be used for turning lights on .

Block Diagram

Main Used Hardware

  • Atmega 328 MicroController
  • RC522 RFID Reader
  • ESP8266 WIFI Module
  • HC-SR04 UltraSonic sensors
  • HC-SR501 PIR Motion Detector

Used Communication Protocols

  • UART
    • Inteface with ESP8266 WIFI Module
  • SPI
    • Inteface with RC522 RFID Reader
